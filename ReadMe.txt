Pour utiliser API nous avons besoin de la base de données « dbvoisins.sql » que nous avons joint au document, 
ainsi que des applications XAMPP et Postman qui ont été utilisé pour faire les différents tests.

Pour démarrer l’API il faut copier le dossier « rest-api-application » dans XAMPP (C:\xampp\htdocs\) puis démarrer  
les modules Apache et MySQL sur le XAMPP control Panel.
